# last_words
Source  : https://www.tdcj.state.tx.us/death_row/dr_executed_offenders.html
The dataset contains following column
- Execution
- Offender_information_Link
- Last_Statement_link
- Last_Name
- First_Name
- TDCJ_Number
- Age
- Date
- Race
- County
- Last_words

You can use for any purpose whatsoever.
